# GIDA Labs: Assessment Investigation 001

**Assessment Investigation 001: When does assessment measure knowledge, and when does it measure access?**

This prototype was designed for EDU-3381: Diverse Learners in the Mainstream Classroom.

## Purpose

This interactive micro professional-development module demonstrates how multilingual learners may appear to struggle academically when assessment conditions create barriers unrelated to the intended learning target.

The content target is intentionally simple: plants need water, sunlight, and air to grow. Across the case files, the science concept remains stable while the access conditions change.

## Learning Objectives

Participants will:

1. Distinguish between content knowledge and assessment access.
2. Consider how language access, visual access, and communication mode affect assessment validity.
3. Explain why multiple evidence sources are necessary when assessing multilingual learners.
4. Connect classroom assessment decisions to the work of Elizabeth Coelho, Jim Cummins, Ofelia García, and Margo Gottlieb.

## Important Limitation

The visual-access case is not intended to simulate dyslexia, visual impairment, or any specific disability. It is an access-barrier demonstration that supports discussion about assessment validity and the risk of missed or mistaken identification for multilingual learners.
